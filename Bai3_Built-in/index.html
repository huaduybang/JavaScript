Một số built-in được căn bản trong JS
alert
console
prompt
confirm
set timeout
set interval

+ alert // xuất ra một cảnh báo
+ console // in ra một thông báo...
+ confirm // in ra 1 thông báo kèm nút ok và cancel...
+ prompt // in ra 1 thông báo kèm ô input...
+ set timeout // thực hiện code sau một thời gian cán đích...
+ set interval // thực hiện sau 1s
